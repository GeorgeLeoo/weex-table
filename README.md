## weex做的表格，支持头部固定，列固定，横向滚动 

注： 本项目适用于eros搭建的weex项目，如果用官方weex脚手架搭建的项目请集成bindingx，将 this.$bindingx 改成你的 bindingx

## 使用了bindingx做交互，简单的实现了头部固定，列固定，横向滚动


## 使用：
将 table.vue 下载下来，替换放到您的项目中使用即可
